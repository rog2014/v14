Assets Tracking

The purpose of this module is to track available assets with Departments/ Employee in the company

So, the management knows each asset with which employee.

There are 2 roles for this module:
  • Manager
  • User

The difference between the previous roles is that the manager can see the finical information while the user could not.

Features
  • Truck assets by responsible officer inside the company.
  • Manager user can track the assets with financial information related to the assets (Gross Value- Salvage Value- Residual Value)
  • User can track assets without financial information.
Installation

This module is insalled normally like any other module

Configuration

There is no special configuration for this module

Usage

1- Assign proper role for the employee



2- Navigate to Asset Menu in Apps menu.



3- From assets list choose the one which need to be tracked



4- Fill required information for the assets like: Responsible Employee, S/N, Department
